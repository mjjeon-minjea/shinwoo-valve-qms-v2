## 주요 변경 사항

### 1. 세부 분석현황 페이지 버그 수정
- `ProcessAnalysis.jsx`, `WorkplaceAnalysis.jsx`, `EquipmentAnalysis.jsx`, `ModelCategoryAnalysis.jsx` 내 데이터 매핑 및 필터 오류 수정.
- 필터 변경 시 현재 달을 기준으로 정상 작동하도록 기본 날짜 설정 로직 개선.

### 2. 공정검사 대시보드 (`ProcessInspectionDashboard.jsx`) 전면 개편
- 사용자 제공 PPT 시안에 맞추어 UI 전면 재구성.
- 100 PPM (0.01%) 목표치를 하드코딩된 기준으로 설정하여 모든 컴포넌트에 반영.
- 4개의 핵심 요약 KPI 카드 구성.
- 4개의 **반원 도넛 형상 (게이지 차트)** 구현.
    - SVG 해킹 방식에서 **CSS 기반 회전 바늘(Needle)** 방식으로 마이그레이션 변경하여 일관성과 반응성 확보.
    - 값에 비례해 연속적으로 움직이는 바늘이 아닌, **해당 상태 구간(우수/정상/경고/심각)의 정중앙**을 가리키도록 로직 수정.
    - 숫자 값이 바늘 뒤에 가리지 않도록 차트 하단으로 배치 수정.
- 하단 상태 범례 추가 (우수, 정상, 경고, 심각 4단계 구간 설명).
- 우측 품질 관리 상태 뷰 추가로 한눈에 위험 항목 식별 가능하도록 개선.
