# 🎨 이미지 아카이버 규칙 (image-archiver)

본 스킬은 `generate_image` 도구로 임시 생성된 파일들을 보존하고 형상 관리에 누락 없이 통합하기 위한 강제 규정입니다.

## 1. 이미지 생성 시 프로젝트 전용 폴더 복사 규칙 [조항 6]
- **자동 복사 의무**: 에이전트가 `generate_image` 도구를 사용하여 이미지를 생성한 경우, 생성 완료 즉시 본 스킬을 스스로 기동하여 프로젝트 전용 이미지 보관 경로로 이전시켜야 합니다.
- **공식 보관 경로**: 이전할 대상 폴더는 반드시 **`안티그래비티\images\`** 폴더입니다.
- **누락 방지**: 사용자의 별도 지시를 기다리지 않고, 이미지가 생성된 턴에 에이전트가 즉각 책임을 지고 자동 이전 처리를 완수해야 합니다.
