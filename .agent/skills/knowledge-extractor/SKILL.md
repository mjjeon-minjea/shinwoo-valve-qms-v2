# Skill: Knowledge Extractor (knowledge-extractor)

## 1. 동작 원리
본 스킬은 대화 종료 지시 감지 시, `decision_*`, `troubleshoot_*`, `feature_*`, `*_rulebook.md` 4대 분류 기준에 맞추어 지식을 자동 추출하고 다중 분할 저장하는 도구입니다.

## 2. 세부 실행 프로토콜
1. 대화 내역 전체를 파싱하여 채택/기각된 기술 의사결정, 트러블슈팅 코드, 기능 명세, 절대 룰셋을 스캔합니다.
2. 추출된 내용을 절대 섞지 않고, 각각 독립된 물리 마크다운 파일로 분할 저장합니다.
3. 대화에 존재하지 않는 카테고리는 건너뛰지 않고 "해당 사항 없음(N/A)"을 명시하여 보고합니다.
4. 맨 마지막 논의가 오간 과거 날짜(YYYYMMDD)를 파일명 끝에 강제 부여하여 저장합니다.
