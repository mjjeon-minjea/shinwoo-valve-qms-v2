# 📖 QMS 위키 통합 관리 규칙 (qms-wiki-manager)

본 스킬은 신우밸브 QMS 옵시디언 위키의 데이터 정합성 검사, 신규 지식 인제스트, 위키 기반 지식 검색을 관리하는 종합 스킬입니다.

## 1. 위키 인제스트 (wiki-ingest)
- 위키의 `raw/` 디렉토리에 추가된 `pending` 상태의 소스 문서들을 감지하여, 이를 정제된 지식 페이지로 변환해 `wiki/` 디렉토리로 이동시킵니다.
- 인제스트 완료 후 전체 지식 인덱스(`index.md`) 및 변경 이력 로그(`log.md`)를 자동으로 갱신합니다.

## 2. 위키 린트 (wiki-lint)
- 위키 내의 모든 깨진 위키링크(Wikilinks), 인덱스 불일치 여부, 고립된 페이지, 그리고 미인제스트 소스 파일을 주기적으로 검사합니다.
- 린트 검사는 수정 제안만 출력하며, 에이전트가 단독으로 자동 수정을 적용하지 않습니다.

## 3. 위키 질의응답 (wiki-query)
- 차장님이 위키에서 지식 탐색을 지시하시는 경우, `wiki/` 내의 정식 문서를 일차적 출처로 삼아 답변을 구성합니다.
- 대화를 통해 새로 획득한 지식이나 트러블슈팅 결과는 누락 없이 즉각 신규 위키 페이지로 릴리즈하여 지식 플라이휠을 회전시킵니다.
