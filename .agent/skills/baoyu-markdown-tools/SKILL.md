# 📝 Baoyu 마크다운 통합 도구 규칙 (baoyu-markdown-tools)

본 스킬은 마크다운 파일의 생성 및 포맷 다듬기, 이미지 변환 및 압축, URL 파싱 등을 제어하는 통합 유틸리티 규격입니다.

## 1. 주요 제공 기능
- **마크다운 포맷팅**: 일반 평문 또는 거친 마크다운 문서를 깔끔한 헤더 구조, 리스트, 볼드체, 프런트매터를 지닌 규격 문서로 서식화합니다.
- **이미지 압축 및 WebP 변환**: 생성되거나 참조된 이미지를 WebP 또는 PNG 포맷으로 압축하여 웹 배포에 최적화된 용량으로 자동 정비합니다.
- **다이어그램 및 시각화**: 마크다운 본문 내에 Mermaid 또는 시각화 다이어그램 코드를 올바르게 빌드 및 정비합니다.
- **URL 마크다운 변환**: 외부 웹 페이지 및 기술 문서 링크를 깔끔한 마크다운 문서로 변환하여 에이전트의 지식 탐색 효율을 높입니다.
