### 작업 1 — InboundAnalysis.jsx 추출

**대상 파일:** `src/components/Dashboard.jsx`

Dashboard.jsx 안에 인라인으로 정의된 `InboundAnalysis` 함수 컴포넌트를 찾아 별도 파일로 분리한다.

```
할 일:
1. Dashboard.jsx에서 InboundAnalysis 함수 컴포넌트 전체 블록을 잘라낸다
2. src/components/InboundAnalysis.jsx 파일을 새로 만들고 붙여넣는다
3. 필요한 import 문을 InboundAnalysis.jsx 상단에 추가한다
4. Dashboard.jsx 상단에 아래 import를 추가한다:
   import InboundAnalysis from './InboundAnalysis';
5. Dashboard.jsx에서 기존 InboundAnalysis 함수 정의 블록을 삭제한다
```

완료 기준: `npm run dev` 실행 시 인수검사 화면이 정상 렌더링되어야 한다.
